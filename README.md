# A brief description of the Jupyter notebook

The notebook implements the statistical modelling framework and numerical experiments developed in the working paper "Quantifying socio-temporal effects of loan delinquency drivers in microfinance". It also illustrates the implementation of some advanced statistical models and numerical methods using Python, including data simulation, model estimation, and analysis workflows.

To ensure reproducibility while respecting data confidentiality constraints, synthetic datasets are generated directly within the code to mimic the structure of the original data. These datasets are then used to reproduce the estimation procedures and computational experiments discussed in the Appendix C.

The code further provides an implementation of the OMCC, a classification approach proposed in the paper and based on the Matthews Correlation Coefficient (MCC).
